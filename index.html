Blending modes

Showcase of all individual CSS blending modes.

Setup

We have two different backgrounds to properly show the blend. Blending modes are set via css property mix-blend-mode

The property can contain following values:

  • normal
  • multiply
  • screen
  • overlay
  • darken
  • lighten
  • color-dodge
  • color-burn
  • hard-light
  • soft-light
  • difference
  • exclusion
  • hue
  • saturation
  • color
  • luminosity

You can find additional information at Mozilla mix-blend-mode

Modes

Normal

No blending applied each color is perserved and stacked on top of each other.

Multiply

Each RGB channel is represented as float <0,1> and the top layer is multiplied by bottom layer f(a,b) = a*b. When the value is <1.0 the result will be darker color. If the value is=1.0 the top layer is unchanged.
